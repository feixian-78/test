1、自动出入库机制：
仓管员审核采购订单或销售订单通过后，系统自动创建商品入库、出库记录，并自动同步更新商品信息中的库存数量
2、个性化图表展示:
仓管员展示商品信息总数、库存统计图表。采购人员展示个人采购订单总数、采购量统计图表。销售人员展示个人销售订单总数、销售量统计图表

【角色与功能】
管理员（Admin）
后台首页查看各类统计图表，包括商品信息总数、采购订单总数、销售订单总数、商品库存统计、采购量统计、销售量统计
用户管理: 管理管理员、仓管员、销售人员、采购人员的信息(增删改查)
基础信息管理: 管理供应商、仓库、客户的信息(增删改查)
商品信息管理: 新增商品分类、新增商品信息，选择仓库、商品分类、填写商品名称、价格、数量、详情等信息。支持商品的入库、出库操作。
采购入库管理: 查看所有商品的入库信息、采购订单
销售出库管理: 查看所有商品的出库信息、销售订单

仓管员（​​Inventory Manager​​）
登录/注册、后台首页显示商品信息总数、库存统计
商品信息管理: 对商品进行入库、出库操作。填写入库或出库的数量、选择对应的供应商或客户
采购入库管理: 查看所有商品的入库信息、审核采购订单信息
销售出库管理: 查看所有商品的出库信息、审核销售订单信息

采购人员（Buyer）
后台首页显示商品信息管理: 查看所有商品信息
采购入库管理: 新增商品的采购订单，选择商品、填写采购数量、选择供应商，提交审核

销售人员(Salesman)
后台首页显示自己的销售订单总数、销售量统计
商品信息管理: 查看所有商品信息
销售出库管理: 新增商品的销售订单，选择商品、填写数量、选择客户，提交审核

【技术架构】
SpringBoot、Mybatis-Plus、MySQL、Vue、Element-Plus
